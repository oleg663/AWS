Що демонструє цей проект

Git Workflow

Автоматизація при кожному push в main гілку. Практика Git потоку.

GitHub Actions

Налаштування CI/CD пайплайнів без додаткових інструментів.

Безпека

Використання Secrets для SSH ключів та конфіденційних даних.

AWS Infra

Робота з EC2, Security Groups, SSH, та веб-серверами.

Як працює пайплайн

1

Локальна розробка

Розробка на локальному комп'ютері з використанням Git для контролю версій.

2

Push на GitHub

Код пушиться на GitHub, що запускає GitHub Actions workflow.

3

Автоматичний деплой

GitHub Actions підключається до AWS через SSH та деплоїть файли.

4

Сайт оновлено

Nginx автоматично віддає нові файли. Сайт оновлено без downtime.
