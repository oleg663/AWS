Про цей проект

Цей проект створений для практики CI/CD пайплайнів.

Технології:

  • HTML5
  • CSS3
  • JavaScript
  • GitHub Actions
  • AWS EC2
  • Ubuntu Server

Що робить CI/CD пайплайн:

  1. При пуше в репозиторій запускається GitHub Actions
  2. Перевіряється синтаксис HTML/CSS
  3. Файли передаються на AWS сервер через SSH
  4. Nginx віддає статичні файли
  5. Статус деплою відображається на сайті
